# リソース操作・コスト記述仕様

本システムにおける、チャットパレットおよびスキルのコスト欄で使用するリソース操作の記述仕様です。

## 基本構文

リソース（HP, MP, 弾薬など）を操作するためのコマンドは以下の形式で記述します。

```
:[対象リソース][演算子][値]
```

*   **対象リソース**: `HP`, `MP`, `弾薬`, `魔石` など、システムに登録されているリソース名。
*   **演算子**: `+` (回復/増加) または `-` (消費/減少)。
*   **値**: 固定値、または計算式。

### 例

*   `:MP-5` (MPを5消費)
*   `:HP+10` (HPを10回復)
*   `:弾薬-1` (リソース「弾薬」を1消費)

## 動的な値（計算式）

値の部分に `()` を使用することで、計算式を記述できます。計算式内ではステータス値を参照可能です。

```
:[対象リソース][演算子]([計算式])
```

### 例

*   `:HP-(Level*2)` (レベル×2 の値をHPから消費)
*   `:MP-(Grade+5)` (グレード+5 の値をMPから消費)
*   `:Sanity-(1d6)` (1d6のダイスロール結果をSanityから減少) ※ダイスロール対応はシステム依存

## 複数操作の記述

複数のリソースを同時に操作する場合、`;` (セミコロン) または `,` (カンマ)、あるいはスペースで区切って記述します。

### 例

*   `:MP-5; 弾薬-1` (MPを5消費し、かつ弾薬を1消費)
*   `:HP-10, MP-5`

## スキル「コスト」欄での記述

スキルのコスト欄では、先頭の `:` (コロン) を省略して記述することが可能です。

*   `MP-5`
*   `MP-5, 弾薬-1`
*   `HP-(Level*5)`

システムはこれらの記述を解析し、スキル使用時に自動的にリソース操作を実行します。
